windows_in_VN_fix
Scripts and tweaks to fix some common problems for Windows users in Vietnam.

Contents
This repository currently includes several .bat files used for:
Fixing Windows time synchronization errors
Tweaking services and network settings for personal computers

How to use
Download the repository or clone it using Git:
git clone https://github.com/Leneoto-Hinsuay/windows_in_VN_fix.git
Run the required .bat file with Administrator privileges.

Notes
For personal use only, not for businesses/organizations/agencies
Some scripts will change system configuration
Create a Restore Point before use
Read the script content before running

Compatible with:
Windows 10
Windows 11

Purpose
This repository was created to store and share quick fix scripts for Windows users in Vietnam.
